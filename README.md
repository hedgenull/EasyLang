# EasyLang Docs
___



### Comments
> **- `comment`**
> Anything you write after the dash, will be skipped. Can be useful to make your code more readable.



### Printing
> **say `text`**
> This will print `text`. 


### Variables
> **var `VariableName` = `Variable Value`**
